# TravelFlux Pro - Advanced Travel Planning Platform

A sophisticated web application for planning travel routes across Andhra Pradesh with intelligent route optimization, transport comparison, and booking capabilities.

## Features

- **Smart Route Planning**: Calculate exact distances between 125+ cities in Andhra Pradesh
- **Transport Comparison**: Compare bus, train, car, auto, and flight options with pricing
- **City Exploration**: Browse detailed information about cities with images and attractions
- **User Authentication**: Sign up/login system with age verification
- **Booking System**: Complete booking flow with multiple payment options
- **AI Chatbot**: 24/7 travel assistant for queries
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Dark/Light Mode**: Toggle between color themes

## Setup Instructions

1. **Clone or Download** the project files
2. **Open `index.html`** in a modern web browser
3. **No server required** - works as a static website

## Usage Guide

### 1. Planning a Route
- Enter starting city and destination in the planner
- Select travel date and number of passengers
- Click "Find Route" to calculate distance and travel time
- Compare different transport options

### 2. Exploring Cities
- Browse the cities section to explore Andhra Pradesh
- Filter by city type (Major, District, Tourist, Other)
- View city details including images and attractions

### 3. Booking a Trip
- Login or sign up for an account
- Verify age (required for certain transport options)
- Fill passenger details and proceed to payment
- Choose payment method (Card/UPI)

### 4. Using the AI Chatbot
- Click the robot icon in bottom-right corner
- Ask questions about travel, routes, or booking
- Use quick question buttons for common queries

## Technologies Used

- **HTML5**: Structure and semantics
- **CSS3**: Styling with custom black/violet theme
- **JavaScript**: Interactive functionality
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Icons
- **SweetAlert2**: Modern alerts and modals
- **Fuse.js**: Fuzzy search functionality
- **LocalStorage**: Client-side data storage
- **IndexedDB**: Browser database for bookings
